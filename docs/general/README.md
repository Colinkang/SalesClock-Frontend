# 📋 通用文档

## 🎯 项目概述

**拜访管理系统** - 基于 React + Express 的客户拜访管理平台

## 🚀 快速开始

### 开发环境
```bash
# 前端
npm run dev

# 后端
cd backend && npm run dev
```

### 生产部署
```bash
docker-compose up -d
```

## 🔧 技术栈

- **前端**: React + TypeScript + Tailwind CSS
- **后端**: Node.js + Express + Prisma
- **数据库**: PostgreSQL
- **部署**: Docker + Docker Compose
